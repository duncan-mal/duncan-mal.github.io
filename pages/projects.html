ASU FSE100 Project

Oct/21/2025 - Nov/25/2025

Designed and prototyped an assistive cooking device using a Raspberry Pi that enables visually impaired users to safely monitor food temperature. The system integrates a digital temperature probe and converts real-time readings into audio feedback using text-to-speech technology.

Key contributions:
-Interfaced a temperature sensor with Raspberry Pi GPIO
-Developed Python scripts to read and process temperature data
-Implemented text-to-speech output for hands-free accessibility
-Designed the system with a focus on usability and safety

This project emphasized inclusive engineering design, embedded systems integration, and human-centered problem solving.
